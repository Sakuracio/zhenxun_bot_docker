# zxenv

#### 介绍
尝试使用docker快速部署zhenxun_bot的一个方案  

##### 使用方法
请提前下载好docker 国内服务器请设置好容器加速镜像   
```shell
# 请自行使用docker volume或映射目录做好数据持久化运行
# 拉取镜像

docker pull hibikier/zhenxun_bot:版本号

# 不清楚版本号请到Docker Hub查看
```
